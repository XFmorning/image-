# Morning AI
#AI自由画布桌面端

AI 图像生成桌面客户端 (Electron + React + Vite)

## 目录结构

```
MorningAI/
├── Morning AI/                  # 封装后的 Electron 应用（可直接运行）
│   ├── Morning AI.exe           # 应用主程序
│   ├── resources/app/           # 应用源码（React + Vite）
│   └── ...                      # Electron 运行时 & 依赖
├── Morning AI Setup 1.0.0.exe   # Windows 安装包
└── .gitignore
```

## 技术栈

- Electron
- React 19 + React Router
- Ant Design 6
- Vite
